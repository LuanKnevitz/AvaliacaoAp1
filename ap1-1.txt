Git init- Criar um repositório local
Git touch- Criar um arquivo de texto
Git add . - Adicionar os arquivos que vou mandar
Git status - Conferir o que está acontecendo no bash
Git commit -m"" - Descrição e fechamento do pacote que será enviado
Git push --set-upstream origin master - Envia as alterações para o repositório remoto
